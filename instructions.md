# How to load WaitMate in Chrome

1.  Open Chrome and navigate to `chrome://extensions`.
2.  Enable **Developer mode** using the toggle in the top-right corner.
3.  Click the **Load unpacked** button.
4.  Navigate to and select the folder where the files are located (`c:\Users\Rohit\Desktop\waitmate`).
5.  WaitMate is now active! Open any website to see it in action.

### Files included:
- `manifest.json`: Configuration for the extension.
- `content.js`: Logic for the overlay, messages, and progress.
- `styles.css`: Dark theme styling and animations.
- `icon.png`: (Placeholder) The extension icon.

*Note: Since there is no actual `icon.png` provided, Chrome will use a default icon if you don't add one. You can use any 128x128 image and name it `icon.png`.*
